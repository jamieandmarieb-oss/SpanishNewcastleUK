# Spanish Newcastle UK

A cultural and media resource webpage by María Zubelzu de Brown.

## Purpose

This project introduces Spanish learners and university students to newspapers, magazines, YouTube channels, musicians, artists, writers and celebrities from Spain and Latin America. It is designed as a practical gateway for reading, listening, cultural research and independent discovery.

## Countries included

- España
- Argentina
- Chile
- Perú
- Uruguay
- Bolivia
- Colombia
- México
- Puerto Rico
- Costa Rica
- Paraguay

## Features

- Search by country, person, media outlet, description or category
- Country filter
- Category filter
- Expand and collapse country sections
- Responsive design for mobile, tablet and desktop
- Real photographs for each country section
- External cultural and media links
- Accessible and attractive blue card layout with alt text, focus states and semantic HTML
- Back-to-top button

## Data maintenance

External links should be checked regularly because newspapers, magazines, YouTube channels, public profiles and cultural websites may change ownership, structure or URL format. The current dataset favours official websites where available and reputable reference profiles when an official source is not practical.
